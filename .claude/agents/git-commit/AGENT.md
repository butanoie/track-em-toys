# Git Commit Agent

A custom global agent for handling git commits with the following capabilities:

- Analyzes staged changes to generate meaningful commit messages
- Follows conventional commits format
- Validates that documentation requirements are met (per project guidelines)
- Creates concise, focused commit messages
- References relevant issue numbers when applicable
- Ensures commits are atomic and well-scoped
