Reading the design
Primary region
The user's country (from settings) is always shown first at full size. Country badge (e.g., CA) is only rendered when other regions exist.
Regional expando
Appears only if data exists for regions beyond the user's. Shows count: "3 other regions". Applies independently to MSRP and Market Value tiers.
Market Value range bar
Gradient (green → amber → red) indicates price-distribution position. Marker shows where the median sits between the min and max. Compact version in expando shows only min/max numbers and a shorter bar.
Freshness indicator
Green dot = < 7 days (market value) or < 30 days (MSRP). Amber = aging. Red = stale.
"You Paid" row
Only visible when the item is in the user's collection. Dashed top border separates private data from shared catalog data. Amber/earth tone accent on the label distinguishes it from automated pricing.
